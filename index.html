Responsive

JSON Driven

How to Use

  1. Remix: Use our Form Builder to generate your custom event.json.
  2. Download: Fork or clone the repository from GitHub.
  3. Replace: Overwrite the existing event.json with your generated code.
  4. Deploy: Host it for free on GitHub Pages, Vercel, or Netlify.
